# NNotes


First time:

```shell
flutter create --project-name template app
```

## Run

Debug mode (only chrome is supported to debug web applications)

```shell
cd app
flutter run -d chrome --web-port 3000 # seems for debugging only chrome is supported
```

Release mode

```shell
cd app
flutter run -d chrome --web-port 3000 # seems for debugging only chrome is supported
```
